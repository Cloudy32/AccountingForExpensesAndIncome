Задача: Учет расходов
    Создай приложение для учета расходов, которое позволяет пользователю добавлять, удалять и просматривать свои расходы.

Основные требования:
    Функциональность:
        Пользователь может добавлять расходы.
        Пользователь может удалять расходы.
        Пользователь может просматривать все расходы с общей суммой.
Объектно-ориентированный подход:
    Создай класс Expense, представляющий отдельный расход.
    Создай класс ExpenseTracker, который управляет списком расходов.
Библиотеки:
    Используй библиотеку datetime для работы с датами.
    Используй библиотеку prettytable для форматированного отображения расходов в виде таблицы.
    Если ты знаком с pandas, можешь использовать его для обработки данных.
Архитектура приложения:
    Раздели приложение на модули:
        models.py — классы для моделирования данных.
        views.py — функции для отображения информации пользователю.
        main.py — основной файл запуска программы, который взаимодействует с пользователем.
